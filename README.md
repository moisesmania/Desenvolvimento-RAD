# SGA - Sistema de Gerenciamento de Alunos

## Requisitos
- Delphi 12
- MySQL
- phpMyAdmin
- FireDAC

## Banco de Dados
Execute o arquivo banco.sql.

## Configuração padrão
Server=127.0.0.1
Database=Prova
User_Name=root
Password=
DriverID=MySQL

## Funcionalidades
- Projeto MDI
- CRUD completo
- FireDAC + MySQL
- Pesquisa com parâmetro e macro
- REST API funcional
